# CSSの基本形

## 基本の型と各部の名称

基本型

- どの部分の
- 何を
  - `{}` で囲む
- どうする

```css
/* <p>タグのフォントサイズを20pxにする */

p { font-size: 20px; }

/* 
p セレクタ
font-size プロパティ
20px 値
*/
```

## セレクタ

**どの部分の** にあたる

- 要素セレクタ
- 全称セレクタ
- 子孫セレクタ
- 複数セレクタ
- idセレクタとclassセレクタ
- 属性セレクタ

## プロパティ

**何を** にあたる

## 値

**どうする** にあたる

## 複数のプロパティを書く場合

開業して縦に並べる

```css
p {
  color: #ef626c;
  font-size: 16px;
  margin: 20px;
}
```
