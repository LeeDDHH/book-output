# レスポンシブWebデザインを理解する

## Webサイトを閲覧する環境の変化

レスポンシブWebデザイン、**1ソースマルチユース**  
1つのHTMLファイルをCSSファイルで制御することで、端末の画面サイズに合わせたデザインとして表示する

メリット

- 1つのHTMLファイルの情報をベースに端末適応する
- 個別に更新する必要がない
- 情報の差異が生まれにくい

デメリット

- 1ソースであるため、モバイルへPCと同じ重さのデータを読み込ませてしまう
- 端末ごとの細かな調整が難しい
