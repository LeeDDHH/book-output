# Webページの見た目や配置を指定するCSS

正しいHTMLコーディングができてから、Webページの**わかりやすい見た目**や**配置**を表示する

## Webページのデザインやレイアウトをお行うCSS

HTMLのみだと、白背景に黒文字の味気ないデザインで表示される  
**Webユーザービリティ**観点で、**わかりやすく情報を伝える**ために**デザイン**や**レイアウト**をする

CSSを別ファイルにするメリット

- Webページの文書構造とデザイン、レイアウトに関する記述が分離できる
  - 文書構造　→　HTML
  - デザイン、レイアウト　→　CSS

## CSSで行うことができる新しい表現

CSSの解釈はWebブラウザの種類/バージョンによって異なる

CSS記述がどのWebブラウザでサポートされているのかは[Can I use... Support tables for HTML5, CSS3, etc](https://caniuse.com/)で確認する
