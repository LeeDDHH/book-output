# ボーダーの指定

## ボーダーを指定するプロパティ

- `border-width`
  - px
  - em
- `border-style`
  - solid
  - dotted
  - dashed
  - double
- `border-color`

## ショートハンド

```markdown
border: 4px solid #000;

border: ボーダーの幅 ボーダーの形 ボーダーの色
```

## ボーダーを1辺のみ/1辺ずつ指定する

## ボックスを角丸にする

`border-radius`
  - 角丸の半径を指定
  - 初期値は0
  - pxで指定する
